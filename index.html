About

예배 현장에서 바로 쓰는
PPT 자동화 도구

WorshipPPT는 실제 교회 예배 환경에 맞게 설계된 Windows 데스크톱 프로그램입니다. PowerPoint COM 자동화를 기반으로 찬양·찬송·성경 PPT를 빠르게 구성하고, 모바일 미러링으로 성도 스마트폰에 실시간 동기화합니다.

WorshipPPT is a Windows desktop application built for Korean church worship services. It automates PowerPoint slide management for praise songs, hymns, and Bible verses, with a mobile mirroring interface for real-time congregation display.

Features

주요 기능

🎵

찬양 PPT 관리

찬양 슬라이드를 PPTX 형식으로 생성·편집. 악보 이미지, 내비게이션 점, 텍스트박스 레이아웃 자동 구성.

📖

성경 / 찬송가 검색

어절 단위 한국어 검색으로 성경 구절·찬송가를 빠르게 찾아 슬라이드에 삽입.

📱

모바일 미러링

같은 Wi-Fi 환경에서 iOS·Android 브라우저로 현재 슬라이드를 실시간 동기화. PWA 오프라인 캐시 지원.

⚡

PowerPoint 직접 제어

COM 자동화로 PowerPoint를 직접 제어. 슬라이드 전환, 예배 시작·종료, ESC 키오스크 모드 지원.

💾

JPEG 슬라이드 캐시

모든 슬라이드를 JPEG로 미리 내보내 모바일 로딩 속도를 최적화.

🗂️

예배 순서 통합 관리

찬양·설교 자막·특순·광고 등 예배 전체 PPT를 한 화면에서 구성하고 순서대로 제어.

Workflow

사용 흐름

예배 PPT 준비

찬양·설교 자막 슬라이드를 찬양 검색 또는 직접 편집으로 생성.

슬라이드 내보내기

JPEG 캐시를 생성해 모바일 미러링 속도 최적화.

예배 시작

PowerPoint를 키오스크 모드로 실행, 본 화면에 투영.

모바일 동기화

성도·음향 담당자 스마트폰에서 현재 슬라이드 실시간 확인.
